Sobre o Curso

Em março de 2026, um pacote npm publicado pela Anthropic acidentalmente incluiu o código-fonte completo do Claude Code — 512 mil linhas de TypeScript que revelaram a arquitetura interna do agente de IA mais avançado do mercado.

Pense assim: se o modelo de linguagem (Claude) é o chef, o Claude Code é a cozinha inteira — com receitas, utensílios, regras de segurança e logística. Até o vazamento, só conhecíamos os pratos. Agora podemos estudar como a cozinha funciona por dentro.

Este curso transforma essa oportunidade única em aprendizado estruturado. Ao longo de 3 trilhas, 12 módulos e 72 tópicos, você vai entender como o Claude Code gerencia conversas, orquestra ferramentas, coordena sub-agentes, protege o sistema e muito mais — conhecimento prático que vai transformar a forma como você trabalha com agentes de IA.
